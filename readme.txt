== Description ==
Este plugin muestra una burbuja de WhatsApp flotante en el sitio, permitiendo que los visitantes contacten rápidamente a través de WhatsApp.

== Installation ==
1. Sube el archivo .zip desde el panel de administración de WordPress.
2. Activa el plugin desde el menú 'Plugins'.
3. Configura tu número de WhatsApp en la página de ajustes.
